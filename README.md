# vigenere

## Usage
Using **vigenere** is really simple. Below is how program read parameters:
```
vigenere <command> [input] [key]
```
### Encrypt
To encrypt use command `e`:
```
vigenere e "Hello" "Key"
```
The output should be:
```
Rijvs
```

### Decrypt
To decrypt use command `d`:
```
vigenere d "Diqd" "Key"
```
The output should be:
```

```
